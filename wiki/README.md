# PhishLens-TR Wiki İçeriği

Bu klasördeki Markdown dosyaları GitHub Wiki'ye aktarılabilir.

## GitHub Wiki'ye Aktarma

1. GitHub reponuzda **Wiki** sekmesini açın
2. **Create the first page** veya mevcut sayfayı düzenleyin
3. Bu klasördeki `.md` dosyalarının içeriğini kopyalayıp ilgili sayfalara yapıştırın

## Dosya Eşlemesi

| Dosya | Wiki Sayfası |
|-------|--------------|
| Home.md | Ana sayfa (Home) |
| Kurulum.md | Kurulum |
| Yapilandirma.md | Yapılandırma |
| Ozellikler.md | Özellikler |
| CLI-Referansi.md | CLI Referansı |
| Sorun-Giderme.md | Sorun Giderme |
| _Sidebar.md | Sidebar (sol menü) |

## Not

`_Sidebar.md` içeriği Wiki'nin sol menüsünde görünür. GitHub Wiki'de bu sayfayı "Sidebar" olarak adlandırın.
